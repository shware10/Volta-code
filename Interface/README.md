# Interface

게임 내 주요 상태 변화 이벤트를 전달하기 위한 인터페이스 모음입니다.

## Scripts

- **IHpListener**  
  HP 변경 시 호출되는 이벤트 인터페이스

- **IQuickSlotChangedListener**  
  퀵슬롯 전체 데이터가 변경될 때 호출되는 인터페이스

- **IQuickSlotSelectedListener**  
  퀵슬롯 선택이 변경될 때 호출되는 인터페이스

- **IStaminaListener**  
  스태미너 값 변경 시 호출되는 인터페이스

- **ISwapListener**  
  퀵슬롯 간 아이템 위치 스왑 시 호출되는 인터페이스
