# StatusData

플레이어의 상태 데이터(체력, 스태미너, 킬 수, 인벤토리 등)를 관리하는 스크립트 모음입니다.

## Scripts

* **HpManager**
  플레이어 체력, 데미지 처리, 사망 및 탈출 로직을 관리하는 클래스 

* **StaminaManager**
  플레이어 스태미너 소모 및 회복을 상태에 따라 관리하는 클래스 

* **KillManager**
  플레이어 킬 수를 네트워크 기반으로 관리하는 클래스 

* **Quickslot**
  아이템 슬롯 추가, 제거, 스왑 등 인벤토리 기능을 관리하는 클래스
