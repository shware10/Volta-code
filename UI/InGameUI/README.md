# InGameUI

게임 플레이 중 UI 상호작용 및 상태 표시를 담당하는 스크립트 모음입니다.

## Scripts

* **DraggableUI**
  슬롯 아이템을 드래그하여 이동할 수 있도록 처리하는 UI 클래스 

* **DroppableUI**
  드래그된 아이템을 받아 슬롯 간 교환을 처리하는 UI 클래스 

* **PointerButtonInteract**
  버튼에 마우스 hover 및 클릭 시 시각적 피드백을 처리하는 클래스 

* **QuickslotView**
  퀵슬롯 전체 UI를 갱신하고 선택 상태를 표시하는 뷰 클래스 

* **SlotView**
  개별 슬롯의 아이템 표시 및 선택 상태를 관리하는 클래스 

* **StatusView**
  체력, 스태미너, 플레이어 이름 등을 UI에 표시하는 클래스 

* **Timer**
  게임 시작 전 카운트다운을 UI로 표시하고 이벤트를 발생시키는 클래스
